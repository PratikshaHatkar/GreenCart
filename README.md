#  GreenCart – Full Stack Grocery E-Commerce Website

GreenCart is a full-stack grocery e-commerce web application built using the MERN stack. It provides a seamless online grocery shopping experience with secure authentication, product browsing, shopping cart management, online payment integration, and an seller dashboard for product management.

##  Features

###  User

* User registration and login with JWT authentication
* Browse grocery products by category
* Search and filter products
* Add and remove items from the shopping cart
* Place orders securely
* Online payment integration using Razorpay
* View order history
* Responsive design for desktop and mobile devices

###  Seller

* Secure seller login
* Add new grocery products
* Update product stock

##  Tech Stack

### Frontend

* React.js
* JavaScript
* HTML5
* CSS3
* Axios

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* bcrypt.js
* Multer
* Cloudinary
* Razorpay

##  Authentication

* JWT-based authentication for users
* Protected routes for authenticated users
* Secure seller dashboard with authorized seller access

##  Key Highlights

* Full-stack MERN architecture
* RESTful API development
* Secure authentication and authorization
* Cloud image storage with Cloudinary
* Online payment gateway integration
* Responsive user interface
* Environment-based configuration for secure deployment
